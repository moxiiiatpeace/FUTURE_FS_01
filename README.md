# Personal Professional Portfolio Website

## Overview

This project is a Personal Professional Portfolio Website developed to showcase my skills, projects, education, and professional profile in an interactive and user-friendly way. The website serves as a digital portfolio that highlights my technical abilities and provides an easy way for visitors to learn more about me and connect with me.

## Features

* Responsive and modern website design
* Interactive Home section
* About Me section
* Skills section
* Project showcase section
* Contact form
* Smooth navigation between sections
* SEO-friendly structure
* Simple and clean UI

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Sections

### Home

Displays introduction and professional title.

### About

Contains educational background and career interests.

### Skills

Shows technical skills and technologies.

### Projects

Displays projects and work samples including:

* PYQ Hub
* Calculator App
* Student Management System

### Contact

Allows users to send messages and connect.
